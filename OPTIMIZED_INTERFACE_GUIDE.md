## 🎯 使用流程

### 1. 快速开始
```bash
python seeed_robocore_server.py
# 打开 seeed_robocore_studio_optimized.html
```

### 2. 设备发现
1. 点击左侧 "🔍 重新扫描"
2. 确认发现的设备列表
3. 点击设备选中进行控制

### 3. 内存表格操作 (核心功能)
1. 切换到 "内存表格" 标签
2. 使用搜索框快速定位地址
3. 使用类型过滤器过滤显示
4. 点击 "读" 按钮读取当前值
5. 点击 "写" 按钮修改RW类型地址

### 4. 实时控制
1. 选择控制模式 (手动/实时)
2. 拖动滑块或输入数值
3. 使用快速预设按钮
4. 观察右侧实时日志

### 5. 设备标定
1. 点击 "⚙️ 设备标定"
2. 选择标定类型:
   - 🏷️ **ID配置**: 重新分配舵机ID
   - 🌡️ **温度限制**: 设置保护阈值
   - 🎯 **中位校准**: 手动精确校准

## 🚀 高级使用技巧

### 🔍 内存表格搜索技巧
- **地址搜索**: 输入 "0x13" 或 "13"
- **名称搜索**: 输入 "temperature" 匹配相关地址
- **描述搜索**: 输入 "电压" 找到电压相关地址

### 🏷️ ID管理最佳实践
1. **统一编号**: 使用连续数字 (1,2,3...)
2. **分类管理**: 不同类型设备使用不同范围
3. **记录映射**: 记录ID与物理位置对应关系

### 🎯 中位校准技巧
1. **安全第一**: 务必先禁用扭矩
2. **精确校准**: 舵盘完全朝上时保存
3. **验证测试**: 保存后测试移动效果

### 📋 操作日志分析
- **成功日志**: [success] 标识确认操作成功
- **错误日志**: [error] 红色标识需要处理的问题
- **警告日志**: [warning] 黄色标识注意事项
- **信息日志**: [info] 蓝色标识一般信息

## 🔧 故障排除

### 📊 内存表格问题
- **读不到值**: 检查设备连接和ID选择
- **写入失败**: 确认地址权限为RW
- **值不更新**: 检查EEPROM锁定状态

### 🎮 控制问题
- **滑块无响应**: 确认选择了实时模式
- **移动不准确**: 检查参数范围和单位
- **无反馈**: 查看右侧实时日志

### 📱 连接问题
- **设备未发现**: 检查硬件连接和供电
- **服务器断开**: 查看左侧连接状态
- **延迟较高**: 检查串口和波特率
