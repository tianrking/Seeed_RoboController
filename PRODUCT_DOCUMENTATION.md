# Seeed RoboCore Studio - 产品文档

## 🎯 产品概述

**Seeed RoboCore Studio** 是由Seeed Studio打造的专业机器人控制平台，为lerobot项目和机器人开发者提供现代化、易用的上位机解决方案。

### 🌟 核心价值
- **易用性优先**: 新手友好的界面，5分钟即可上手
- **专业功能**: 高级用户可深度配置所有参数
- **标定导向**: 针对新设备的初次使用场景优化
- **产品化设计**: 现代化的用户体验和工业级稳定性

## 🚀 快速开始

### 系统要求
- Python 3.7+
- 现代Web浏览器 (Chrome, Firefox, Edge, Safari)
- FTServo舵机及专用电源
- USB转串口适配器

### 安装启动

1. **启动服务器**
   ```bash
   python seeed_robocore_server.py
   ```

2. **自定义端口 (可选)**
   ```bash
   python seeed_robocore_server.py --port COM9 --baudrate 1000000
   ```

3. **打开Web界面**
   在浏览器中打开 `seeed_robocore_studio.html`

### 首次使用流程

对于lerobot项目用户，推荐按以下顺序操作：

1. **设备连接** → 确保舵机正确连接和供电
2. **扫描设备** → 点击"扫描设备"发现连接的舵机
3. **开始标定** → 点击"开始标定"进行设备标定
4. **实时控制** → 使用实时控制功能进行操作

## 🎯 核心功能

### 1️⃣ 实时控制功能

#### 🎮 双模式控制系统

**🎯 手动模式**
- 传统数值输入 + 按钮点击模式
- 适合精确定位场景
- 每次操作前可确认参数

**⚡ 实时模式 (推荐)**
- 拖动滑块实时控制舵机
- 毫秒级响应反馈
- 直观的用户体验

#### 🎛️ 控制参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 位置 | 0-4095 | 舵机角度位置 |
| 速度 | 0-2047 | 运动速度 |
| 加速度 | 0-255 | 运动平滑度 |

#### 🎯 快速预设
提供8个常用角度快速按钮：
- 0°, 45°, 90°, 135°, 180°, 225°, 270°, 315°

### 2️⃣ 设备标定功能

#### 📋 标定流程

**步骤1: 设备连接**
- 确保舵机正确连接
- 扫描并选择设备

**步骤2: ID配置**
- 设置1-253范围内的唯一ID
- 建议使用连续编号

**步骤3: 温度限制**
- 设置50-100°C温度保护阈值
- 防止设备过热损坏

**步骤4: 中位校准**
- 调整舵机到物理中位
- 确保舵盘朝上精度

**步骤5: 完成标定**
- 查看标定摘要
- 开始使用设备

#### 🎯 标定要点
- **安全性**: 温度限制保护设备
- **精确性**: 中位校准确保精度
- **规范性**: ID管理便于维护

### 3️⃣ 高级功能

#### ⚙️ PID参数调节
- **位置PID**: P、I、D参数独立调节
- **速度PID**: P、I参数优化运动效果
- **启动扭矩**: 设备启动特性配置

#### 💾 内存访问
- 直接读写舵机内存地址
- 支持所有64个逆向工程地址
- 快速访问常用地址

#### 📊 固件信息
- 固件版本查询
- 硬件版本信息
- 设备识别和兼容性

## 🔄 WebSocket API

### 连接信息
- **地址**: `ws://localhost:8765`
- **协议**: WebSocket + JSON
- **认证**: 本地访问无需认证

### 基础消息格式
```json
{
    "type": "消息类型",
    "参数名": "参数值"
}
```

### 核心API列表

#### 设备管理
```javascript
// 扫描设备
{
    "type": "scan_servos",
    "start_id": 1,
    "end_id": 20
}

// 获取设备状态
{
    "type": "get_servo_status",
    "servo_id": 1
}
```

#### 运动控制
```javascript
// 移动舵机
{
    "type": "move_servo",
    "servo_id": 1,
    "position": 2048,
    "speed": 100,
    "acceleration": 50
}

// 停止舵机
{
    "type": "stop_servo",
    "servo_id": 1
}
```

#### 标定配置
```javascript
// 设置温度限制
{
    "type": "write_temperature_limit",
    "servo_id": 1,
    "temp_limit": 75
}

// 写入PID参数
{
    "type": "write_pid_params",
    "servo_id": 1,
    "position_pid": [32, 0, 32],
    "speed_pid": [10, 200]
}
```

#### 高级功能
```javascript
// 内存访问
{
    "type": "read_memory",
    "servo_id": 1,
    "address": 63,
    "length": 1
}

// 固件信息
{
    "type": "get_firmware_info",
    "servo_id": 1
}
```

## 🔧 故障排除

### 常见问题

#### 连接问题
**问题**: 无法连接服务器
- 检查服务器是否启动 (看控制台输出)
- 确认端口8765未被占用
- 检查防火墙设置

#### 硬件问题
**问题**: 扫描不到设备
- 检查串口名称是否正确
- 确认设备供电正常
- 检查USB连接线

**问题**: 舵机不响应
- 确认舵机ID配置正确
- 检查波特率设置 (默认1000000)
- 验证设备供电电压

#### 控制问题
**问题**: 实时控制不响应
- 确保在"实时模式"
- 检查设备是否被选中
- 验证滑块参数范围

### 调试技巧

#### 查看日志
- 服务器控制台显示详细日志
- Web界面显示操作日志
- 错误信息通常包含解决建议

#### 网络调试
- 打开浏览器开发者工具
- 查看WebSocket连接状态
- 检查消息发送和接收

#### 硬件调试
- 使用官方工具验证硬件
- 检查串口通信参数
- 验证电源供电稳定性
