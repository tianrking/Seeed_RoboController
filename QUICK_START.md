# Seeed RoboCore Studio - 快速启动指南

## 🚀 5分钟快速上手

### 步骤1: 启动服务器
```bash
# 进入项目目录
cd diy_tool/

# 启动服务器
python seeed_robocore_server.py
```

看到以下输出表示启动成功：

### 步骤2: 打开Web界面
在浏览器中打开：`seeed_robocore_studio.html`

### 步骤3: 新设备标定 (首次使用必需)

1. **点击"开始标定"**
2. **扫描设备** → 发现连接的舵机
3. **配置ID** → 设置舵机唯一标识
4. **设置温度限制** → 保护设备安全
5. **中位校准** → 调整舵盘朝上精度
6. **完成标定** → 开始使用

### 步骤4: 实时控制
- 选择"⚡ 实时模式"
- 拖动位置滑块实时控制舵机
- 使用预设按钮快速定位

## 🔧 常用命令

### 自定义端口启动
```bash

```

## ❓ 常见问题

**Q: 扫描不到设备怎么办？**
A: 检查串口名称、设备供电、USB连接

**Q: 实时控制不响应？**
A: 确保选择了舵机并启用"实时模式"

**Q: 如何连接多台设备？**
A: 系统支持多个舵机，每个需要唯一ID
